centralのアクション
home トップページ

search 検索アクション　ヘッダーに埋め込む


userアクション
new ユーザー登録画面
create ユーザー登録システム

share 画像共有

pictures　全ての絵の一覧
destroy 共有停止

sessionアクション
create ログイン
